# 总结

做这些题的时候，有些特意用了慕课教的数据结构的定义，有些则是用 STL 代替偷懒了，因为有方便的东西不用真的很心塞嗷（比如优先队列代替小顶堆大顶堆，map 代替散列查找，STL 的东西可好使）。
不管咋样咕咕咕到了最后一刻总算是做完了，这个超～漫长的暑假算没白过 23333，寒假再接再厉加把劲准备学 java~ 完结撒花【bushi】
