思路及代码

动态规划，设 dp 为第i台阶有多少种跳法，由于每次可以跳一层或二层，则 dp[i] = dp[i-1]+dp[i-2]（从 i-1 跳一层，从 i-2 跳两层）。其实就是上题斐波那契数列，只不过第一项由0变为1罢了。

  • 用 min 记录当前最小值，ans 记录最大利润
